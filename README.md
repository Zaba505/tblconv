# tblconv

A simple tool for converting one "table" format into another.

## Supported Formats

* CSV
  - [x] source
  - [x] output
* Excel
  - [x] source
  - [x] output
* SQL
  - [x] source
  - [x] output

## CLI Features

The `tblconv` CLI supports all of the above formats. Some peculiarities for each
format are the following.

### Built-in SQL Drivers

- [x] Postgres
- [x] MySQL
- [ ] Excel
- [ ] CSV
